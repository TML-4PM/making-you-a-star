# Interview Story & Job Analysis Platform

A comprehensive React-based application for managing interview stories, analyzing job descriptions, and preparing for interviews using AI-powered insights.

## 🚀 Features

### Core Functionality
- **Story Management**: Import, organize, and optimize interview stories using the STAR method
- **Job Analysis**: AI-powered analysis of job descriptions to extract key themes and requirements
- **Interview Preparation**: Generate personalized flashcards and practice sessions
- **Story Optimization**: Enhance story quality with AI-driven feedback and scoring
- **Bookmarking System**: Save and organize important stories and job analyses

### AI Integration
- OpenAI GPT-4 powered story analysis and optimization
- Automatic theme extraction from job descriptions
- Intelligent story-to-job matching recommendations
- Quality scoring and improvement suggestions

### User Experience
- Clean, responsive design with dark/light mode support
- Intuitive navigation and search functionality
- Real-time data synchronization
- Mobile-optimized interface

## 🛠 Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS with custom design system
- **Backend**: Supabase (PostgreSQL, Authentication, Edge Functions)
- **AI**: OpenAI GPT-4 API
- **State Management**: React Query for server state
- **UI Components**: shadcn/ui component library
- **Icons**: Lucide React
- **File Processing**: XLSX for Excel import/export

## 🚦 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Supabase account

### Installation

1. **Clone the repository**
   ```bash
   git clone <YOUR_GIT_URL>
   cd <YOUR_PROJECT_NAME>
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
